# 🎯 About This Repository

Welcome!

This repository documents my complete cybersecurity learning journey—from learning how computers and networks work to becoming a professional penetration tester.

Rather than simply collecting notes, this repository serves as my personal cybersecurity knowledge base, practical lab notebook, project portfolio, and certification tracker.

Everything I learn from books, courses, labs, documentation, and real-world practice will be organized here by topic.
