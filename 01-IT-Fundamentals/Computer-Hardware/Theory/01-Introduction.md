# Overview

Computer hardware refers to the physical components of a computer system. These are the parts that can be seen and touched, and they work together to receive input, process data, store information, and produce output.

Every computer, whether it is a desktop, laptop, server, smartphone, or embedded device, is built from hardware components. Understanding these components is the foundation of Information Technology (IT) and Cybersecurity.

Before learning how to secure computers or networks, it is essential to understand how computers are built and how they operate.

# What is Computer Hardware?

Computer hardware is the collection of all physical components that make up a computer system.

Unlike software, which consists of programs and instructions, hardware is the equipment that performs the actual work.

Examples include:

- Central Processing Unit (CPU)
- Motherboard
- Memory (RAM)
- Storage Devices
- Power Supply Unit (PSU)
- Graphics Processing Unit (GPU)
- Keyboard
- Mouse
- Monitor
- Printer

Without hardware, software cannot run.

Without software, hardware cannot perform useful tasks.

Both depend on each other.

---

# Hardware vs Software

| Hardware | Software |
|----------|----------|
| Physical components | Programs and applications |
| Can be touched | Cannot be touched |
| Manufactured | Developed |
| CPU, RAM, SSD | Windows, Linux, Microsoft Word |

Hardware provides the platform.

Software provides the instructions.

---

# Major Categories of Hardware

Computer hardware can be grouped into several categories.

## Processing Hardware

Responsible for executing instructions and performing calculations.

Examples:

- CPU
- GPU

---

## Memory Hardware

Temporarily stores data that is actively being used.

Examples:

- RAM
- Cache Memory

---

## Storage Hardware

Stores information permanently even when power is turned off.

Examples:

- HDD
- SSD
- USB Flash Drive

---

## Input Devices

Allow users to send information to the computer.

Examples:

- Keyboard
- Mouse
- Scanner
- Webcam
- Microphone

---

## Output Devices

Present information from the computer to the user.

Examples:

- Monitor
- Printer
- Speakers
- Projector

---

# How Hardware Works Together

Every computer follows the same basic process.

```text
Input Device
      │
      ▼
CPU Processes Data
      │
      ▼
RAM Stores Working Data
      │
      ▼
Storage Saves Data
      │
      ▼
Output Device Displays Result
```

Although each component has a different responsibility, they constantly communicate to perform tasks efficiently.

# Why Learning Hardware is Important

A strong understanding of computer hardware is essential for anyone pursuing a career in IT or Cybersecurity.

Hardware knowledge helps with:

- Building and upgrading computers.
- Troubleshooting hardware failures.
- Installing operating systems.
- Creating virtual lab environments.
- Understanding system performance.
- Diagnosing computer problems.
- Supporting users and organizations.

Hardware is one of the core foundations upon which networking, operating systems, cybersecurity, and penetration testing are built.

---

# Hardware and Cybersecurity

Cybersecurity professionals work with computer hardware every day.

Understanding hardware helps when:

- Performing digital forensic investigations.
- Securing workstations and servers.
- Building penetration testing labs.
- Recovering damaged systems.
- Investigating hardware failures.
- Understanding trusted hardware technologies such as TPM and Secure Boot (covered later).

Although cybersecurity often focuses on software and networks, every attack ultimately affects physical hardware.

---

# Real-World Example

Imagine opening a web browser.

The following sequence occurs in a fraction of a second:

1. You click the mouse.
2. The mouse sends an input signal.
3. The CPU processes the request.
4. RAM temporarily stores the running application.
5. The storage device loads browser files.
6. The monitor displays the browser window.

This cooperation between hardware components allows computers to perform complex tasks quickly and efficiently.

# Key Takeaways

- Computer hardware consists of all physical components of a computer.
- Hardware and software depend on each other.
- Hardware can be categorized into processing, memory, storage, input, and output devices.
- Every hardware component has a specific responsibility.
- Understanding hardware is the first step toward becoming an IT and cybersecurity professional.
